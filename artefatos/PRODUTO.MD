# Diagnóstico de Produto - Calculadora do Cidadão (BCB)

## 1. Melhorias de Interface e UX
* **Interface Responsiva:** O site atual não é adaptado para dispositivos móveis, o que dificulta o uso fora do desktop.
* **Unificação de Fluxos:** Atualmente, cada índice (IPCA, SELIC, TR) exige que o usuário navegue por menus diferentes. Uma interface única com seleção de índice via "dropdown" reduziria o número de cliques.

## 2. Sugestões de Novas Funcionalidades
* **Cálculo Comparativo:** Permitir que o usuário veja a correção de um mesmo valor por diferentes índices simultaneamente para tomada de decisão.
* **Processamento em Lote:** Criar uma funcionalidade nativa para upload de arquivos (como o CSV que usamos na automação) para processar múltiplos cálculos de uma vez.

## 3. Oportunidades com Inteligência Artificial (IA)
* **Linguagem Natural (NLP):** Um chat onde o usuário digita: "Quanto vale 1.500 reais de janeiro de 2025 hoje pelo IPCA?" e a IA preenche o formulário sozinha.
